# Titanic Data Cleaning and Preprocessing

## Objective

The objective of this project is to clean and preprocess the Titanic dataset to make it suitable for further analysis and machine learning tasks.

## Dataset

Dataset: Titanic Dataset

File Used: train.csv

Number of Rows: 891

Number of Columns: 12

## Tools and Technologies

* Python
* Pandas
* NumPy
* VS Code
* Jupyter Notebook

## Steps Performed

### 1. Data Loading

The Titanic dataset was loaded using Pandas.

### 2. Data Inspection

Dataset structure, shape, and missing values were examined.

### 3. Missing Value Handling

* Age missing values were filled using the median value.
* Embarked missing values were filled using the mode value.
* Cabin column was removed due to a large number of missing values.

### 4. Duplicate Removal

Duplicate records were checked and removed if present.

### 5. Data Type Conversion

Relevant columns were converted to appropriate data types.

### 6. Column Renaming

Some column names were renamed for better readability.

### 7. Exporting Clean Data

The cleaned dataset was saved as cleaned_titanic.csv.

## Results

* Missing values successfully handled.
* Dataset cleaned and prepared for analysis.
* Cleaned dataset exported successfully.
